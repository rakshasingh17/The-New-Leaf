The New Leaf
Project Overview

The New Leaf is a frontend-only productivity web app designed specifically for engineering students. It helps users organize academic tasks, track deadlines, manage subjects, view progress, and plan schedules through a clean, distraction-free interface. The app focuses on usability, visual clarity, and academic workflows using interactive UI components and simulated data.

Tech Stack Used

HTML5

CSS3

JavaScript

How to Run the Project

Download or clone this repository.

Open the folder on your computer.

Double-click index.html or open it in any modern web browser.

Navigate through the app using the provided links.

No backend, database, or installation is required.
